# math-project

# 🧮 Matematik Asistanı V1.0

Kullanıcıların kendi matematiksel formüllerini dinamik olarak kaydedip hesaplama yapabilecekleri, aynı zamanda rastgele sorularla pratik yapabilecekleri terminal tabanlı bir Python uygulamasıdır.

## 🚀 Öne Çıkan Özellikler

*   **Dinamik Formül Yönetimi:** `SQLite3` veritabanı entegrasyonu sayesinde kendi formüllerinizi sisteme kaydedebilir ve kalıcı olarak saklayabilirsiniz.
*   **Akıllı Hesaplama Araçları:** Kaydedilen formülleri `x` ve `y` değişkenlerine atayacağınız dinamik değerlerle anında çalıştırabilirsiniz.
*   **Antrenman Modu:** Rastgele üretilen dört işlem sorularıyla (Toplama, Çıkarma, Çarpma) matematik pratiği yapabilir ve puan toplayabilirsiniz.
*   **Raporlama Sistemi:** Antrenman modunda elde ettiğiniz skorları, tarih ve saat damgasıyla birlikte `gunluk_rapor.txt` dosyasına otomatik olarak kaydeder.

## 🛠️ Kullanılan Teknolojiler

*   **Dil:** Python 3.x
*   **Veritabanı:** SQLite3 (Dahili kütüphane)
*   **Dosya Yönetimi:** Yerleşik Python Dosya I/O işlemleri
